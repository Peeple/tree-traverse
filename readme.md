Дан массив, описывающий дерево:

```
$tree[
    $tree = [
    ['id' => '8', 'parent_id' => '6',],
    ['id' => '2', 'parent_id' => '1',],
    ['id' => '3', 'parent_id' => '1',],
    ['id' => '4', 'parent_id' => '1',],
    ['id' => '5', 'parent_id' => '2',],
    ['id' => '1', 'parent_id' => '0',],
    ['id' => '6', 'parent_id' => '2',],
    ['id' => '7', 'parent_id' => '3',],
    ];
```

Требуется вывести дерево в следующем виде

```
1
1.2
1.2.5
1.2.6
1.2.6.8
1.3
1.3.7
1.4
```
